# 📝 Assessment-Platform

## MERN Stack App

A streamlined assessment platform focused on creating and solving multiple-choice tests with printing them automated variation generation.

The system allows administrators to manage topics and questions, while users can take tests on selected topics and receive dynamically generated variations exported as Word documents.

---

## 📦 Tech Stack

### 🎨 Frontend
- React  
- Redux Toolkit  
- Tailwind CSS
- Lucide-React
- Framer Motion
- QrCode
- DocX
- JsPdf

### ⚙️ Backend
- Node.js  
- Express  
- MongoDB  
- JWT  
- Cloudinary
- Multer

---

## 🧠 Features

- 🔐 **Authentication**
  - JWT-based login and registration    

- 🛡 **Role-Based Access**
  - Admin and User roles  

- 📚 **Topic-Based Question System**
  - Topics with multiple-choice questions **only**
  - Structured question organization  

- 📝 **Test Solving**
  - Users can take tests based on available topics 
  - Simple and intuitive test-taking flow online

- 🔄 **Test Variation Generation**
  - Generate multiple versions of tests  
  - Randomized questions and answer options  

- 📄 **Word Export**
  - Export generated test variations as Word documents
